# Base de Datos - CocaCola EC Web Replica

## Autor
Kevin Yalama

## Fecha
08/04/2026

## Descripción
Esta carpeta contiene la estructura de la base de datos del proyecto final de Ambientes Computacionales. La base de datos está diseñada para soportar un sistema web de catálogo y compras inspirado en Coca-Cola Ecuador.

## Archivos

### schema.sql
Contiene la estructura de tablas del sistema:
- usuarios
- categorias
- productos
- carrito
- carrito_items
- pedidos
- pedido_items

### seed.sql
Contiene datos iniciales de prueba:
- categorías de bebidas
- productos del catálogo

## Objetivo
Permitir el almacenamiento y gestión de:
- usuarios
- productos
- compras
- pedidos
- carrito de compras
